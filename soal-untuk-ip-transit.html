26. Selesaikan soal dibawah ini! a. 192.168.0.0/26 Tentukan Subnet Mask, Host, IP Host awal, IP Host Akhir, IP Broadcast! b. 192.168.0.1/29 Tentukan Subnet Mask, Host, IP Host awal, IP Host Akhir, IP Broadcast!​di kerjakan di selembar kertas lah kk


a. 192.168.0.0/26

Subnet Mask = 255.255.255.192

Host = 62

IP Host Awal = 192.168.0.1

IP Host Akhir = 192.168.0.62

IP Broadcast = 192.168.0.63

b. 192.168.0.1/29

Subnet Mask = 255.255.255.248

Host = 6

IP Host Awal = 192.168.0.1

IP Host Akhir = 192.168.0.6

IP Broadcast = 192.168.0.7

Pembahasan

Subnetting dilakukan untuk mengelola jaringan dalam sebuah konfigurasi, contohnya di sebuah kantor, untuk membagi jaringan tersebut pada beberapa jalur seperti jalur di beberapa bagian kantor tersebut. Selain itu, tujuan lain dari subnetting dari sebuah jaringan adalah optimalisasi dan efisiensi kerja jaringan, karena jalur lalu lintas tidak terpusat di satu network besar, tapi terbagi ke beberapa ruas-ruas jaringan.

Dynamic Host Configuration Protocol (DHCP) merupakan service yang memungkinkan perangkat dapat mendistribusikan/assign IP Address secara otomatis pada host dalam sebuah jaringan. Cara kerjanya, DHCP Server akan memberikan response terhadap request yang dikirimkan oleh DHCP Client.

Selain IP Address, DHCP juga mampu mendistribusikan informasi netmask, Default gateway, Konfigurasi DNS dan NTP Server serta masih banyak lagi custom option (tergantung apakah DHCP client bisa support).

Mikrotik dapat digunakan sebagai DHCP Server maupun DHCP Client atau keduanya secara bersamaan. Sebagai contoh, misalnya kita berlangganan internet dari ISP A. ISP A tidak memberikan informasi IP statik yang harus dipasang pada perangkat kita, melainkan akan memberikan IP secara otomatis melalui proses DHCP.
