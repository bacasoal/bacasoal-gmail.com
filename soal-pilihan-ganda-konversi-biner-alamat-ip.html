1. Tolong konversikan alamat IP 192.168.8.12 ke dalam bentuk biner menggunakan langkah-langkahnya


Jawaban:

1000 0000 . 1010 1000 . 0000 1000 . 0000 1100

Penjelasan:

IP Address jadikan menjadi 32 bit, dengan dibagi 4. Jadi masing-masing 8 bit bilangan biner

192/2 = 96 sisa 0

96/2 = 48 sisa 0

48/2 = 24 sisa 0

24/2 = 12 sisa 0

12/2 = 6 sisa 0

6/2 = 3 sisa 0

3/2 = 1 sisa 1

1/2 = 0 sisa 1

Jadi 192 bilangan binernya adalah 1100 0000

168/2 = 84 sisa 0

84/2 = 42 sisa 0

42/2 = 21 sisa 0

21/2 = 10 sisa 1

10/2 = 5 sisa 0

5/2 = 2 sisa 1

2/2 = 1 sisa 0

1/2 = 0 sisa 1

Jadi 158 bilangan binernya adalah 1010 1000

8/2 = 4 sisa 0

4/2 = 2 sisa 0

2/2 = 1 sisa 0

1/2 = 0 sisa 1

Jadi 8 bilangan binernya adalah 0000 1000

12/2 = 6 sisa 0

6/2 = 3 sisa 0

3/2 = 1 sisa 1

1/2 = 0 sisa 1

Jadi 12 bilangannya binernya adalah 0000 1100

Kesimpulan: 192.168.8.12 dalam bentuk biner adalah 1000 0000 . 1010 1000 . 0000 1000 . 0000 1100

7. 1.konversikan bilangan heksa desimal B6A ¹⁶ ke desimal adalah 2. konversikan bilangan heksa desimal D4 ¹⁶ ke biner adalah 3. konversikan kode biner 1110 0111² ke heksa desimal adalah4. jika diketahui alamat memori suatu data ada di 0005 :00A7 dimanakah lokasi dari alamat fisiknya? 5. jika suatu data memiliki alamat fisik 00 BB pada hardisk, manakah yang mungkin merupakan alamat memorinya​


Jawaban:

Konversikan bilangan heksa B6A ke desimal:

B = 11 (dalam desimal)

6 = 6 (dalam desimal)

A = 10 (dalam desimal)

B6A = 11 x 162 + 6 x 161 + 10 x 160 = 2962

Konversikan bilangan heksa D4 ke biner:

D = 13 (dalam desimal)

D4 = 11010100 (dalam biner)

Konversikan biner 1110 0111 ke heksa desimal:

1110 = 14 (dalam desimal)

0111 = 7 (dalam desimal)

1110 0111 = E7 (dalam heksa desimal)

Jika diketahui alamat memori suatu data ada di 0005:00A7, lokasi fisiknya bisa ditemukan dengan cara mengonversikan alamat memori tersebut dari biner ke desimal, lalu membandingkannya dengan tata letak memori pada komputer.

Jika suatu data memiliki alamat fisik 00 BB pada hardisk, alamat memorinya mungkin bisa ditemukan dengan mengonversikan alamat fisik tersebut dari heksa ke biner, lalu membandingkannya dengan tata letak memori pada komputer. 14. 1. diketahui ip address : 203.202.101.10 tentukan : a. konversi ip tersebut ke bilangan biner (32 bit)! b. jenis kelas dari ip tersebut dan jelaskan! c. network id dan host id ! d. berapa subnet mask default dari ip tersebut!


Jawaban ringkas:

1. a. Hasil konversinya ialah:

11001011.11001010.01100101.00001010

b. termasuk IP kelas C

c. Network ID = 203.202.101

Host ID = 101

d. Subnet mask default = 255.255.255.0

Penjelasan panjang:

a. Bila dipisahkan, hasil konversinya yaitu:

203 = 11001011

202 = 11001010

101 = 1100101

10 =  1010

Karena yg diinginkan adalah konversi biner 32 bit, maka masing-masing angka di atas harus ditulis dg 8 digit. Bila digabung menjadi:

11001011.11001010.01100101.00001010

Cara perhitungan konversinya bisa dilihat di bagian lampiran.

b. Perlu diketahui bahwa IP Address memiliki kelas jaringan sesuai rentang alamatnya, antara lain:

1 - 126 : IP Kelas A

128 - 191 : IP Kelas B

192 - 223 : IP Kelas C

Karena soal di atas memperlihatkan 203 (sebelum titik pertama), maka dapat disimpulkan bahwa IP Address tersebut termasuk dalam rentang kelas C

c. Pada IP kelas C, bagian yg menjadi network ID ialah 3 angka pertama yg masing-masingnya dipisahkan tanda titik.

Sementara host ID ialah angka terakhir yg tampil. Jadi kesimpulannya:

Network ID = 203.202.101

Host ID = 10

d. Subnet mask default IP kelas C yaitu 255.255.255.0

30. konversikan bilamgan biner tersebut dalam bentuk ip address format decimal 11000011.11001011.01111101.00011001​


Jawaban singkat:

195.203.125.25

Penjelasan tambahan:

Perhitungan konversi biner ke decimal:

11000011 = 1x2⁷+1x2⁶+0x2⁵+0x2⁴+0x2³+0x2²+1x2¹+1x2⁰

11000011 = 128+64+0+0+0+0+2+1

11000011 = 195

Jadi 11000011 (biner) = 195 (decimal)

Perhitungan konversi biner ke decimal:

11001011 = 1x2⁷+1x2⁶+0x2⁵+0x2⁴+1x2³+0x2²+1x2¹+1x2⁰

11001011 = 128+64+0+0+8+0+2+1

Jadi 11001011 (biner) = 203 (decimal)

Perhitungan konversi biner ke decimal:

01111101 = 0x2⁷+1x2⁶+1x2⁵+1x2⁴+1x2³+1x2²+0x2¹+1x2⁰

01111101 = 0+64+32+16+8+4+0+1

01111101 = 125

Jadi 01111101 (biner) = 125 (decimal)

Perhitungan konversi biner ke decimal:

00011001​ = 0x2⁷+0x2⁶+0x2⁵+1x2⁴+1x2³+0x2²+0x2¹+1​x2⁰

00011001​ = 0+0+0+16+8+0+0+1

00011001 = 25

Jadi 00011001 (biner) = 25 (decimal)
