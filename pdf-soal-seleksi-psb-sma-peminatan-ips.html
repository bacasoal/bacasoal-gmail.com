3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0, HP = ?

_______________________________________

JAWABAN:

HP = { x | x > -1 }Opsi (B) x > -1

PENJELASAN:

⇔ Eksponen adalah bentuk bilangan berpangkat atas suatu bilangan basis yang berarti bilangan basis dikalikan dengan bilangan basis sebanyak pangkatnya.

⇔ Sifat-sifat eksponen yang digunakan:

aᵇ ⁺ ⁿ = aᵇ . aⁿ(aᵇ)ⁿ = aᵇ * ⁿaᵇⁿ = (aⁿ)ᵇ

PEMBAHASAN SOAL:

⇒ 3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0

Selesaikan menggunakan sifat eksponen:

3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 03²ˣ . 3 + 8 . 3ˣ - 3 > 0(3ˣ)² . 3 + 8 . 3ˣ - 3 > 0

Substitusikan n = 3ˣ:

n² . 3 + 8n - 3 > 03n² +  9n - n - 3 > 03n(n + 3) - (n + 3) > 0(n + 3)(3n - 1) > 0n + 3 = 0 V 3n - 1 = 0n + 3 - 3 = 0 - 3 V 3n - 1 + 1 = 0 + 1n = -3 V 3n : 3 = 1 : 3n = -3 V n = 1/3

Pisahkan menjadi a > 0, b > 0 dan a < 0, b < 0:

n > -3n > 1/3HP = { n > 1/3 }n < -3n < 1/3HP = { n < -3 }HP = { n > 1/3 V n < -3 }

Masukkan nilai n = 3ˣ:

n = { n > 1/3 V n < 3 }3ˣ = { n > 1/3 V n < 3 }3ˣ < -33ˣ > 1/3

Selesaikan pertidaksamaan pertama:

3ˣ < -3Fungsi pemangkatan selalu positif, jika tidak maka penyelesaian bukan bilangan real.x ∉ R

Selesaikan pertidaksamaan kedua:

3ˣ > 1/33ˣ > 3⁻¹x > -1

Diperoleh:

x > -1, makaHP = { x | x > -1 }

40. Sosiolagi peminatan untuk sma semester 2 , soal latihan akhir semester


Sinopsis

Buku ini disusun dan dirancang untuk mengembangkan sikap, pengetahuan, dan keterampilan siswa tentang berbagai fenomena sosial yang ada di masyarakat.

Fitur-fitur unggulan:
1. Nilai karakter berisi karakter yan gdapat dikembangkan setelah mempelajari materi yang dibahas pada suatu bab.
2. Peta Konsep,berisi sejumlah konsep penting yang akan dipelajari dan perlu dipahami siswa.
3. Kata Kunci, istilah-sitilah penting yang berkaitan dengan materi yang perlu diketahui siswa.
4. Apersepsi, pengantar materi di setiap awal bab untuk membangkitkan rasa keingintahuan siswa terhadap materi yang akan dibahas.
5. Aktivitas Mandiri dan Evaluasi, untuk menguji pemahaman materi yang telah dipelajari.
6. Terapan Sosio dan Porto Sosio, mengajak siswa untuk menerapkan berbagai pengetahuan sosiologi yang telah dipelajari melalui berbagai kegiatan.
7. Glosarium dan Indeks, memperkenalkan konsep-konsep sosiologi yang memudahkan siswa menemukan pembahasan suatu konsep.
