2. Diketahui persamaan kuadrat mx^2 – (2m-3)x + (m-1) = 0. Nilai m yang menyebabkan akar-akar persamaan kuadrat tersebut real dan berbeda adalah ... (Ujian Nasional 2013)


Ingat!
Untuk kedua akar real yang berlainan maka D > 0

b² - 4ac > 0
(2m - 3)² - 4(m)(m - 1) > 0
4m² - 12m + 9 - 4m² + 4 > 0
-12m + 13 > 0
-12m > -13
m < 13/12

Jadi, nilai m yang memenuhi adalah m < 13/12.

diket = a =1 b= -5 c = -3
jawab..
[tex] \alpha + \beta = \frac{-b}{a} = \frac{-(-5)}{1} = 5[/tex]
[tex] \alpha . \beta = \frac{c}{a} = \frac{-3}{1} = -3[/tex]

misal
[tex] \alpha -3 = x_{1} [/tex]
[tex] \beta -3 = x_{2} [/tex]
maka
[tex] x_{1} + x_{2} = \alpha + \beta -6[/tex] [tex]= 5 -6 = -1[/tex]
[tex] x_{1} . x_{2} = \alpha \beta - 3 ( \alpha + \beta ) + 9[/tex]
subtitusikan
[tex] = -3 -3(5) +9 = -9[/tex]

susun persamaan kuadrat barunya
[tex] x^{2} - ( x_{1}+ x_{2} )x +( x_{1} . x_{2} ) = 0[/tex]
[tex] x^{2} - (-1) x -9 = x^{2} + x -9 = 0[/tex]

jadi persamaan barunya adalah
[tex] x^{2} +x -9 =0[/tex]

Ingat!
Untuk kedua akar real atau nyata D ≥ 0

b² - 4ac ≥ 0
(m-2)² - 4.1.(2m-4) ≥ 0
m² - 4m + 4 - 8m + 16 ≥ 0
m² - 12m + 20 ≥ 0
(m - 10)(m - 2) ≥ 0

Jadi, batas-batas nilai m yang memenuhi adalah m ≤ 2 atau m ≥ 10.
