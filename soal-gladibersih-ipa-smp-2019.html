gradien garis a:

[tex]m1 = \frac{4}{3}[/tex]

garis a dan b tegak lurus maka:

[tex]m1 \times m2 = - 1 \\ m2 = - \frac{3}{4} [/tex]

persamaan garis b melalui titik (0,3):

[tex]y - y1 = m2(x - x1) \\ y - 3 = - \frac{3}{4} (x - 0) \\ y - 3 = - \frac{3}{4} x \\ y = - \frac{3}{4} x + 3[/tex]

titik potong garis b dengan sumbu X --->y=0:

[tex]y = - \frac{3}{4} x + 3 \\ 0 = - \frac{3}{4} x + 3 \\ - 3 = - \frac{3}{4} x \\ x = - 3 \times ( - \frac{4}{3} ) = 4[/tex]

titik potong (4,0)

(Dikurangi untuk menghilangkan B)

-7C = - 875.000

C = 125.000

Masukkan ke salah 1 persamaan

3B + 2×125.000 = 550.000

3B+250.000 = 550.000

3B = 300.000

B = 100.000

Harga 2B+3C =2×100.000 + 3×125.000

= 200.000+375.000

= Rp575.000
