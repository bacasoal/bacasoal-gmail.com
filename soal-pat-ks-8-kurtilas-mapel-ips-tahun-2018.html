Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4

22. apa itu wujud kebudayaan mapel : ips kls : 8


wujud kebudayaan merupakan tanda dari sebuah kebudayaan nya yg asli
Wujud orang dapat dibedakan menjadi tiga bagian yaitu:


1.  Wujud Gagasan 

Budaya dalam wujud ide / ide ini bersifat abstrak dan ada dalam alam pikiran setiap warga yang tidak bisa diraba atau difoto.



Sistem berfikir yang dibuat oleh orang-orang yang bertanggung jawab dan sangat diperlukan dan didukung. Gagasan-konsep inilah yang membuat hasil karya manusia berdasarkan sistem nilai, cara berfikir dan pola tingkah laku. Wujud budaya dalam bentuk sistem inilah yang juga disebut sistem nilai budaya.


2. Wujud Perilaku (Aktivitas)

Budaya dalam wujud perilaku berpola menurut ide / gagasan yang ada. Wujud perilaku ini merupakan konkrit yang bisa dilakukan dan difilm. Contoh: Petani sedang bekerja di sawah, orang sedang menari dengan permata gemulai, orang sedang berbicara dan lain-lain. Masing-masing aktivitas tersebut berada dalam satu sistem tindakan dan tingkah laku.


3. Wujud Benda Hasil Budaya

Semua benda hasil karya tersebut konkrit, dapat diraba dan difoto. Kebudayaan dalam wujud konkrit ini disebut lembaga fisik. Contoh: Bangunan-Bangunan megah seperti piramida, barang cina, menhir, alat rumah tangga seperti kapak perunggu, gerabah dan lain-lain. 


Dalam masalah sehari-hari wujud itu yaitu gagasan, perilaku dan hasil budaya tidak terpisahkan dan saling mempengaruhi.
