7. lakukan konveksi bilangan berikutsoal jurusan TKJ ​


Jawab:

a. 110₍₁₀₎ = 6E₍₁₆₎b. 1111110₍₂₎ = 7E₍₁₆₎c. 89340₍₁₀₎ = 256374₍₈₎d. 653₍₈₎ = 110101011₍₂₎e. 110100101010101₍₂₎ = 64525₍₈₎f. ABC₍₁₆₎ = 5274₍₈₎

Penjelasan dengan langkah-langkah:

a. Aturan konversi bilangan berbasis 10 (desimal) ke bilangan berbasis 16 (heksadesimal):

- bagi bilangan desimal dg angka 16.

- bila langsung diketahui, hasilnya akhirnya dikonversi sesuai tabel sistem bilangan.

Caranya bisa dilihat di lampiran (sebelah tabel sistem bilangan)

b. Aturan Konversi bilangan berbasis 2 (biner)  ke bilangan berbasis 16 (heksadesimal):

- kelompokkan bilangan biner menjadi 4 digit (dari kanan)

- ubah bilangannya sesuai tabel sistem bilangan.

Sehingga caranya: 1111110 dikelompokkan menjadi sebagai berikut:

0111₍₂₎ = 7₍₁₆₎

1110₍₂₎ = E₍₁₆₎

Jadi 1111110₍₂₎ = 7E₍₁₆₎

c. Aturan konversi bilangan berbasis 10 (desimal) ke bilangan berbasis 8 (oktal):

- bagi bilangan desimal dg angka 8.

- tulis hasilnya dan hasil sisa baginya

- bila hasil akhirnya 7 atau kurang dari 7, hentikan pembagian

- penulisan konversi dari bawah (hasil akhir) ke atas (sisa pembagian)

Caranya bisa dilihat di lampiran (sebelah tabel sistem bilangan)

d. Konversi oktal ke biner dilakukan dg langsung mengubahnya sesuai tabel sistem bilangan, namun 1 digit oktal dituliskan 3 digit biner. Caranya

6₍₈₎ = 110₍₂₎

5₍₈₎ = 101₍₂₎

3₍₈₎ = 011₍₂₎

Jadi 653₍₈₎ = 110101011₍₂₎

e. Konversi biner ke oktal dilakukan dg mengelompokkan biner menjadi 3 digit (dari kanan), lalu mengubahnya ke oktal. Caranya:

110₍₂₎ = 6₍₈₎

100₍₂₎ = 4₍₈₎

101₍₂₎ = 5₍₈₎

010₍₂₎ = 2₍₈₎

101₍₂₎ = 5₍₈₎

Jadi 110100101010101₍₂₎ = 64525₍₈₎

f. Konversi heksadesimal ke oktal dilakukan dg aturan:

- tiap digit bilangan heksadesimal diubah ke biner

- 1 digit bilangan heksadesimal dituliskan 4 digit bilangan biner

- kelompokkan hasil konversi biner tadi menjadi 3 digit (dari kanan)

- ubah biner ke oktal sesuai tabel sistem bilangan. Caranya:

A₍₁₆₎ = 1010₍₂₎

B₍₁₆₎ = 1011₍₂₎

C₍₁₆₎ = 1100₍₂₎

Maka ABC₍₁₆₎ = 101010111100₍₂₎ dikelompokkan menjadi :

101₍₂₎ = 5₍₈₎

010₍₂₎ = 2₍₈₎

111₍₂₎ = 7₍₈₎

100₍₂₎ = 4₍₈₎

Jadi ABC₍₁₆₎ = 5274₍₈₎

Amatilah tabel sistem bilangan di lampiran.

18. Cara penghitungan IP subnetting


jumlah subnet, jumlah host per subnet, blok subnet, alamat host dan broadcast yang valid

Jumlah Subnet = 2x, dimana x adalah banyaknya binari 1 pada oktet terakhir subnet mask (2 oktet terakhir untuk kelas B, dan 3 oktet terakhir untuk kelas A). Jadi Jumlah Subnet adalah 22 = 4 subnet

Jumlah Host per Subnet = 2y – 2, dimana y adalah adalah kebalikan dari x yaitu banyaknya binari 0 pada oktet terakhir subnet. Jadi jumlah host per subnet adalah 26 – 2 = 62 host

Blok Subnet = 256 – 192 (nilai oktet terakhir subnet mask) = 64. Subnet berikutnya adalah 64 + 64 = 128, dan 128+64=192. Jadi subnet lengkapnya adalah 0, 64, 128, 192.

Bagaimana dengan alamat host dan broadcast yang valid? Kita langsung buat tabelnya. Sebagai catatan, host pertama adalah 1 angka setelah subnet, dan broadcast adalah 1 angka sebelum subnet berikutnya.

Untuk mengetahui subnet mask, cukup dilihat dari prefiksnya, dalam hal ini yaitu /16. itu artinya ada 16 deret angka satu dari 32 bit IP (dimulai dari oktet paling kanan). maka subnet mask nya adalah:

Jika dalam biner: 11111111.11111111.00000000.00000000

Jika dalam desimal:
255.255.0.0

Artinya, dengan IP yang diberikan (178.26.0.0) dan dengan subnet mask tersebut, IP tersebut dapat menampung host sebanyak 2 pangkat 16 dikurang 2 (diluar network id dan broadcast id)

25. apa itu IP route? bagaimana cara menghitung atau menentukannya?​


Jawaban:

IP route (Internet Protocol route) adalah suatu aturan atau petunjuk yang digunakan dalam jaringan komputer untuk menentukan jalur atau rute yang harus diambil oleh paket data untuk mencapai tujuan yang ditentukan. IP route digunakan dalam protokol jaringan seperti IP (Internet Protocol) untuk mengarahkan paket data melalui jaringan yang kompleks.

Cara menghitung atau menentukan IP route melibatkan beberapa faktor, termasuk topologi jaringan, tabel routing, dan algoritma routing yang digunakan. Berikut adalah langkah-langkah umum untuk menghitung atau menentukan IP route:

1. Identifikasi tujuan: Tentukan alamat tujuan paket data yang ingin dikirim.

2. Periksa tabel routing: Periksa tabel routing pada perangkat jaringan (seperti router) untuk mencari entri yang sesuai dengan alamat tujuan. Tabel routing berisi informasi tentang jaringan yang dapat dijangkau dan jalur yang harus diambil untuk mencapai tujuan tersebut.

3. Pilih jalur terbaik: Jika ada beberapa entri yang sesuai dengan alamat tujuan, gunakan algoritma routing yang digunakan (misalnya, algoritma shortest path) untuk memilih jalur terbaik. Jalur terbaik biasanya ditentukan berdasarkan faktor seperti jarak, kecepatan, atau biaya.

4. Konfigurasi router: Setelah jalur terbaik ditentukan, konfigurasikan router atau perangkat jaringan lainnya untuk mengarahkan paket data melalui jalur tersebut. Ini melibatkan pengaturan tabel routing dan pengiriman paket data ke tujuan melalui jalur yang telah ditentukan.

Penting untuk dicatat bahwa penghitungan atau penentuan IP route dapat lebih kompleks dalam jaringan yang lebih besar dan kompleks. Dalam kasus tersebut, protokol routing dinamis seperti OSPF (Open Shortest Path First) atau BGP (Border Gateway Protocol) digunakan untuk secara otomatis menghitung dan memperbarui tabel routing berdasarkan perubahan dalam jaringan.

35. cara membuat program untuk menghitung dan mencetak IP seorang mahasiswa


#include<iostream.h>

#include<stdlib.h>

#include<string.h>

//——— Awal dari Class mhs ———//

//Declarasi class mhs

class mhs

{

private:

char nama[24];

int jml_mk,nilai[24],sks[24];

public:

mhs();

void input_data();

void nama_mhs(char *);

int jum_matkul();

int jum_sks();

float ip();

};

//Constructor

mhs::mhs()

{

int i;

strcpy(nama,”NULL”);

jml_mk=0;

for(i=0;i<24;i++)

{

nilai[i]=5;

sks[i]=0;

}

}

//Method untuk memasukkan data ke dalam class

void mhs::input_data()

{

int i;

char bf[30];

//Masukkan nama mahasiswa

cout << “Nama Mahasiswa = “;

cin.getline(nama,24);

//Masukkan jumlah mata kuliah

while((jml_mk<1)||(jml_mk>24))

{

cout << “Jumlah mata kuliah = “;

cin.getline(bf,30);

jml_mk=atoi(bf);

if((jml_mk<1)||(jml_mk>24))

cout << “Masukkan angka 1-24\n\n”;

}

for(i=0;i<jml_mk;i++)

{

//Masukkan nilai untuk masing-masing mata kuliah

while((nilai[i]<0)||(nilai[i]>4)||(strlen(bf)>1))

{

cout << “\nNilai mata kuliah ” << (i+1) << ” = “;

cin.getline(bf,30);

nilai[i]=(int)bf[0];

if(nilai[i]<70) nilai[i]=69-nilai[i];

else nilai[i]=101-nilai[i];

if((nilai[i]<0)||(nilai[i]>4)||(strlen(bf)>1))

cout << “Masukkan huruf A-E atau a-e\n”;

}

//Masukkan jumlah sks untuk masing-masing mata kuliah

while((sks[i]<1)||(sks[i]>4))

{

cout << “Jumlah SKS mata kuliah ” << (i+1) << ” = “;

cin.getline(bf,30);

sks[i]=atoi(bf);

if((sks[i]<1)||(sks[i]>4))

cout << “Masukkan angka 1-4\n\n”;

}

}

}

//Method untuk menampilkan nama

void mhs::nama_mhs(char *s)

{

strcpy(s,nama);

}

//Method untuk menampilkan jumlah mata kuliah

int mhs::jum_matkul()

{

return (jml_mk);

}

//Method untuk menghitung jumlah sks

int mhs::jum_sks()

{

int i,j;

j=0;

for(i=0;i<jml_mk;i++) j+=sks[i];

return(j);

}

//Method untuk menghitung ip

float mhs::ip()

{

int i,j_sks,total;

j_sks=0;

total=0;

for(i=0;i<jml_mk;i++)

{

j_sks+=sks[i];

total+=(nilai[i]*sks[i]);

}

return ((float)total/j_sks);

}

36. Bagaimana caranya agar ip kita tidak sama dengan ip orang lain


Jawaban:

dengan mengubah Alamat IP secara Manual

Penjelasan:

Sebenarnya alamat IP kita ketika terhubung ke suatu jaringan akan automatis terisi (default) dan itu tidak mungkin sama dengan milik orang lain.

tapi kita bisa mengubah alamat IP kita akan tetapi memiliki resiko yaitu tabrakan IP yang terjadi karna ada 2 alamat IP dengan alamat yg sama dalam 1 jaringan.

cara mengubah alamat IP secara manual:

1. masuk ke Control Panel

2. klik Network And Internet

3. klik Open Network and Sharing Center, dan klik device atau modem pada bagian Connections (yg ada logo sinyalnya)

4. di wifi status klik Properties

5. kemudian cari internet Protocol versi 4 (IPv4) dan klik Properties

6. pada internet Protocol versi 4 (IPv4) Properties bagian general, ganti centang menjadi "use the following IP address" kemudian isi IP address, subnetmask, dan default gateway. cara mengetahui Alamat IP kita pada jaringan yaitu melalui CMD ketik ipconfig disitu keliatan Alamat IP, Subnetmask, dan default gateway kita.
