9. itu nomer 7 lks mat kls 11 smt 1, caranya gimana ya?


AB^t = [(1)...(-2)||(3k+2)...(k+1)]

det AB^t = -2
1(k+1) - (3k+2)(-2) = -2
k + 1 + 6k + 4 = - 2
7k + 5 = - 2
7k = - 7
k = - 1

a = 1, 
b = 3k + 2 --> b = 3(-1) -2 = -5
c = -2
d = k + 1 --> d = -1 +1 = 0

[(a...c)||(b)...(d)] = [1...-2||-5...0]
