32. satu kilogram es suhunya -2°C. bila titik lebur es =0°C,kalor jenis es 0,5 kalori g^-1°C^-1,kalor jenis air=1kal g^-1dan 1kalori=4,2J maka kaloe yang diperlukan untuk meleburkan seluruh es tersebut adalah


Diketahui :
m = 1 kg = 1000 gr
To = -2 C
Tt = 0 C
c es = 0,5 kal/grC
c air = 1 kal/grC
L = 80 kal/gr
Ditanyakan : Q untuk meleburkan ?

Dijawab :
Q1 = m x c es x (Tt-To)
      = 1000 x 0,5 x (0-(-2))
      = 1000 x 0,5 x 2
      = 1000 kal

Q2 = m x L
      = 1000 x 80
      = 80.000 kal

Jadi Kalor yg dibutuhkan = 1000 + 80.000 = 81.000 kal
