Pembahasan Jawaban :

pertama, kita lihat terlebih dahulu apa yang diketahui pada soal tersebut :

Dik : F1 = 9N ; F2 = 5N ; F3 = 6N ; F4 = 8N ; r = 0.4

kedua, kita lihat apa yang ditanyakan pada soal :

Dit : Torsi total terhadap poros melalui O (T) ?

ketiga kita jawab dengan menggunakan dengan menggunakan rumus yang kita pelajari tentang dinamika rotasi :

rumus T = F x r

kita hitung terlebih dahulu T1 nya :

T1 = - 9 x 0.4 = - 3.6 Nm
T2 = 5 x 0.4 = 2 Nm
T3 = - 6 x sin 30 x 0.4 = - 1.24 Nm
T4 = 8 x 0 = 0 Nm

jadi, torsi total nya adalah : T1 + T2 + T3 + T4

Torsi Total = (-3.6) + (2) + (-1.24) + 0 =
- 2.84 Nm

jadi, torsi total terhadap poros melalui O adalah - 2.84 Nm

dit : Vp' = ...??
jawab :
P = P'
(Ma + Mb + Mp).Vp = (Ma + Mp).Vp' - Mb.Va
(40 + 40 + 80).(4) = (40 + 80).Vp' - 40.(8)
640 = 120.Vp' - 320
120.Vp' = 960
Vp' = 8 m/s
jadi, kecepatan perahu jadi 8 m/s.
