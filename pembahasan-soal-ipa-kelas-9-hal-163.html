4. mtk semester 2 kelas 7 hal 163 no 12​


Penjelasan dengan langkah-langkah:

diketahui garis g // k, ∠P2 = P3 dan ∠R1 = R2. Jika ∠P1 = 128°, tentukan besar sudut yang lain.

Jawaban :

P2 = (180° - P1) / 2

= (180° - 128°) / 2

= 52° / 2

= 26°

Karena R1 = R2 maka,

R2 = 2 x P2

= 2 x 26°

= 52°

R3 = 180° - (R1 + R2)

= 180° - (52° + 52°)

= 180° - 104°

= 76°

Q = 180° - (P2 + R2)

= 180° - (26° + 52°)

= 180° - 78°

= 102°

Jadi, besar sudut yang lain adalah P2 = 26°, P3 = 26, R1 = 52°, R2 = 52°, R3 = 76°, dan Q = 102°.
