4. Adinda menabung rp 10, 000, 00, dan di setiap minggu berikutnya rp 1000, 00 lebih banyak dari minggu sebelumnya, berapakah jumlah uang tabungannya pada minggu ke-10


Adinda menabung Rp. 10.000 dan di setiap minggu berikutnya Rp. 1000 lebih banyak dari minggu sebelumnya. Maka jumlah uang tabungan adinda pada minggu ke-10 adalah sebesar Rp. 145.000.

Penyelesaian Soal :

Diketahui : Suku pertama (a) = 10.000

                  beda (b) = 1.000

Ditanya : 10 suku pertama (S₁₀) ?

Jawab :

Hitung nilai suku ke-10 (U₁₀) dengan menggunakan cara sebagai berikut :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

Keterangan : Sₙ = Jumlah suku n pertama

                       Uₙ = suku ke-n

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Maka perhitungannya yaitu :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

S₁₀ = [tex]\frac{10}{2}[/tex] (2 (10.000) + (10 - 1) 1.000)

     = 5 (20.000 + 9 (1.000)

     = 5 (20.000 + 9.000)

     = 5 (29.000)

     = 145.000

∴ Kesimpulan jumlah uang tabungan adinda pada minggu ke-10 adalah sebesar Rp. 145.000.

9. Uang Atik + Uang Basir = Rp28.000, 00 Uang Basir + Uang Candra = Rp36.000, 00 Uang Atik + Uang Candra = Rp32.000, 00 Uang Atik + Uang Basir + Uang Candra = ... Pake cara ya


Jawaban:

Misal:

Uang Atik = a

Uang Basir = b

Uang Candra = c

a + b = 28.000

b = 28.000 - a

b + c = 36.000

(28.000 - a) + c = 36.000

28.000 - a + c = 36.000

-a + c = 36.000 - 28.000

-a + c = 8.000

c = 8.000 + a

a + c = 32.000

a + (8.000 + a) = 32.000

a + 8.000 + a = 32.000

2a + 8.000 = 32.000

2a = 32.000 - 8.000

2a = 24.000

a = 24.000/2

a = 12.000

c = 8.000 + a = 8.000 + 12.000 = 20.000

b = 28.000 - a = 28.000 - 12.000 = 16.000

a + b + c = 12.000 + 20.000 + 16.000 = 48.000

Jadi, uang Atik + uang Basir + uang Candra = Rp48.000,00

Berapa kekurangan uang Tari ?

Rp. 1.500 + ........ = Rp 3. 500
