Matematika
Diketahui harga sebuah pensil adalah 2/3 kali harga sebuah spidol, harga sebuah spidol adalah 2/3 kali harga sebuah buku, dan selisih harga sebuah buku dengan sebuah pensil adalah RP1.800,00
Jika harga pensil = a, harga spidol = b, dan harga buku = c, maka
a = 2/3 b
b = 2/3 c
⇔ c = 3/2 b
c - a = 1800
⇔ c = 1800 + a
⇔ 3/2 b = 1800 + 2/3 b
⇔ 3/2 b - 2/3 b = 1800
⇔ 9/6 b - 4/6 b = 1800
⇔ 5/6 b = 1800
⇔ 5b = 10800
⇔ b = 2160
3b = 3 x 2160 = 6480.
Jadi, harga sebuah spidol adalah Rp2.160,00 dan harga 3 buah spidol adalah 
Rp6.480,00
